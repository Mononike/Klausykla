# Laipčiaus Klausykla


**Haptinė garso instaliacija — magistro tezė**

*Laipčiaus Klausykla* tiria klausymąsi kaip kūnišką veiksmą. Lietuviškas žodis **KLAUSYK** paverčiamas taktilių 3D raidžių skulptūromis, kur kiekviena raidė tampa haptine-akustine architektūra, aktyvuojama prisilietimu.

Instaliacija remiasi **kiki-bouba** crossmodaline korespondencija: kampuotos formos generuoja aukšto dažnio, aštrius garsus; apvalios formos — žemo dažnio, šiltus rezonansus. Prisilietimo intensyvumas (per piezo jutiklius) realiu laiku valdo garso parametrus — objektas *kviečia garsą iš savęs*.

Pavadinimas „Laipčiaus Klausykla" yra *metaforos metafora* — instaliacijoje fizinių laiptų nėra.

## Psichoakustiniai efektai

Tai kai smegenys „girdi" daugiau nei yra garso. Pavyzdžiui, du garsai kartu sukuria trečią, kurio realiai nėra. Arba garsas atrodo einantis iš kairės, nors groja iš abiejų pusių. Smegenys nuolat „prideda" prie to ką girdi — ir tai galima kontroliuoti.

## Crossmodalinės korespondencijos

Tai kai vienas pojūtis automatiškai sukelia kitą. Aukštas garsas — smegenys mato kažką aštrų. Žemas garsas — kažką apvalų, sunkų. Tai ne metafora — tai neurologinis faktas. 95% žmonių jaučia tai vienodai, nepriklausomai nuo kultūros.

## Skulptūros

### K — Kiki Cluster

**Forma:** sprogstanti žvaigždė, aštrūs kampai, dygliuota.

**Garsas:** aukšti, aštrūs, daug vienu metu skambančių dažnių.

**Kas vyksta smegenyse:** kai girdi aukštą aštrų garsą ir lieti kampuotą formą — abu pojūčiai sustiprina vienas kitą. Smegenys sako „taip, tai aštru" dviem kanalais vienu metu. Tai nėra sutapimas — tai tas pats signalas dviem keliais.

**Iliuzija:** garsas atrodo dar aštresnis nei yra, nes pirštai „patvirtina" tai ką girdi.

### L — Bass Rumble

**Forma:** sunkus monolitas, platesnis apačioje, grubus paviršius.

**Garsas:** toks žemas, kad daugiau jauti kūnu nei girdi ausimis.

**Kas vyksta:** garsas 25–50 Hz diapazone keliauja per kaulus — tai vadinama kaulų laidumu. Kai laikai sunkų objektą ir jauti vibraciją — smegenys neskiria, ar tai garsas, ar prisilietimas. Du pojūčiai sulimpa į vieną.

**Iliuzija:** objektas atrodo dar sunkesnis nei yra, nes vibracija prideda „svorio" pojūtį.

### A — Modulation & Reverb

**Forma:** susuktas toras, nei vidus nei išorė, dviprasmė.

**Garsas:** plaukiojantis, erdvinis, neįmanoma nustatyti iš kur ateina.

**Kas vyksta:** kai garsas turi daug reverbo ir chorus efektų, smegenys nebegali tiksliai pasakyti „garsas yra čia". Tai sukuria keistą būseną — klausaisi, bet negali identifikuoti. Barthes tai vadino „trečiuoju klausymu" — kai nustoji analizuoti ir pradedi tiesiog būti garse. Forma padeda — ji irgi neturi aiškaus „čia yra priekis".

**Iliuzija:** atrodo, kad garsas sklinda iš pačios formos vidaus, iš tos tuščios erdvės toro viduje.

### U — Low Hum

**Forma:** lygus, organiškas akmuo, minkštas paviršius, norisi apglėbti.

**Garsas:** šiltas, žemas dūzgimas su lėtu vibravimu.

**Kas vyksta:** žemi dažniai su lėtu vibrato smegenims signalizuoja „didelis, apvalus, šiltas". Tai bouba efektas — beveik visi žmonės žemą minkštą garsą sieja su apvaliomis formomis. Kai lieti lygų paviršių ir girdi lygų garsą — atsiranda harmonija, pojūčiai nesipešia.

**Iliuzija:** objektas atrodo šiltesnis nei kambario temperatūra, nes garsas „prideda" šilumos pojūtį.

### S — Sibilant Breath

**Forma:** vingiuota, gyvatės pavidalo, banguota.

**Garsas:** šnabždesys, filtruotas triukšmas, kaip vėjas arba kvėpavimas.

**Kas vyksta:** sibilantai (s, š, z garsai) yra ypatingi — tai ne tonas, o triukšmas, filtruotas per siaurą dažnių juostą. Smegenys juos atpažįsta kaip „gyvą" — kvėpavimą, šnabždesį, vėją. Kai filtras juda aukštyn-žemyn — tai primena gyvatės judesį. Forma ir garsas kartoja tą patį S gestą.

**Iliuzija:** atrodo, kad objektas kvėpuoja, kad jis gyvas.

### Y — Glitch Clicks

**Forma:** beveik taisyklingas daugiakampis, bet kelios viršūnės pasislinkusios.

**Garsas:** spragtelėjimai beveik reguliariu ritmu, bet vis suklumpa.

**Kas vyksta:** smegenys mėgsta ritmą ir nuolat bando jį rasti. Šis garsas beveik duoda ritmą — bet kas kelintą smūgį praleidžia arba paslenka. Tai sukuria mikro-frustraciją: smegenys tikisi smūgio, jo nėra, tada jis ateina netikėtai. Forma kartoja tą patį — atrodo taisyklinga, bet kažkas „ne taip".

**Iliuzija:** dėmesys sustiprėja — negali nustoti klausyti, nes smegenys nuolat bando „išspręsti" raštą, kurio nėra.

## Kiekvienu atveju principas tas pats

Forma ir garsas sako tą patį skirtingais kanalais, ir smegenys juos sujungia į vieną stipresnę patirtį nei kiekvienas atskirai. Tai ne technologijos triukas — tai kaip žmogaus suvokimas veikia. Ši instaliacija tiesiog tai padaro sąmoningai ir kontroliuojamai.

---

## Techninė architektūra

```
[Piezo jutiklis] → [Arduino Uno (A0)] → [USB Serial 115200] → [SuperCollider]
   prisilietimas       analogRead()          println(raw)        garso sintezė
```

Piezo reikšmė (0–1023) normalizuojama į 0.0–1.0 ir valdo:
- **Kurį garsą** groja (švelniai → bouba, stipriai → kiki)
- **Kaip jis skamba** (kiekvienas parametras reaguoja į prisilietimo intensyvumą)

## Aparatūra

**Komponentai:** Arduino Uno, piezo diskas, 1MΩ rezistorius, jungiamieji laidai

```
Arduino A0 ---+--- Piezo (+)
              |
           [1MΩ]
              |
Arduino GND --+--- Piezo (-)
```

## Paleidimas

### SuperCollider (su aparatūra)

1. Įkelk `arduino/piezo_arduino.ino` į Arduino
2. Atidaryk SuperCollider, paleisk `supercollider/klausykla_KLAUSYK.scd` (1 BLOKAS)
3. Paleisk 2 BLOKĄ (serial sujungimas)
4. Paliesi piezo — girdi garsą
5. `~switch.(0)` iki `~switch.(5)` — perjungi skulptūras

### SuperCollider (be aparatūros)

1. Atidaryk ir paleisk `supercollider/klausykla_be_piezo.scd`
2. Naudok GUI slankiklį + mygtukus 1–5

### Web demo (lokaliai)

```bash
cd web-demo
npm install
npm run dev
```

Atsidaro `localhost:5173`.

## Teorinis pagrindas

Projektas remiasi crossmodalinių korespondencijų tyrimais (Ramachandran & Hubbard), įkūnyto pažinimo teorija (Lakoff & Johnson), klausymosi fenomenologija (Barthes), ir psichoakustiniais tyrimais apie garso-formos asociacijas lietuvių kalboje (Mikalonytė & Dranseika).
